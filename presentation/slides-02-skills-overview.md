**Gemini Image Prompt:**
```

Subtitles: "Build" "Verify" "Iterate" "Deploy"

Curved arrow from /resume-it back to /try-it labeled "Add features, fix, test"

Below main flow (smaller, dotted connections up):
"/wrap-it" (slate) → "Save & exit"
"/retrofit-it" (teal) → arrow into /make-it: "Existing apps enter here"
"/argo-it" (coral) → dotted to /ship-it: "K8s deployment"
"/nemo-it" (navy) → "/fix-it" (rose): "Security loop"
```

**Slide Text:**
### The lifecycle: Build -- Verify -- Iterate -- Deploy

**Main flow:**
/make-it → /try-it → /resume-it → /ship-it

**The iterate loop:** Add features with /resume-it, verify with /try-it, repeat until happy.

**Supporting skills:**
- **/wrap-it** -- Save progress, shut down, pick up next session
- **/retrofit-it** -- Existing apps enter the lifecycle here
- **/argo-it** -- Kubernetes deployment path
- **/nemo-it** + **/fix-it** -- Continuous security scanning and auto-fix
