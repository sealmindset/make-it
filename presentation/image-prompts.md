# /make-it Presentation -- Gemini Image Prompts

**Global style for all 10 images:**
- Deep navy background (#0A1628), 16:9 (1920x1080)
- Luminous, translucent, Apple-keynote aesthetic. Soft glows, no drop shadows.
- 95% visual, 5% text. If you can say it with an image, don't use words.
- Palette: blue #3B82F6, green #10B981, amber #F59E0B, coral #EF4444, purple #8B5CF6, white
- No cliches: no icebergs, gears, lightbulbs, rockets, shields, puzzle pieces

## Gemini Generation Notes

- Every image: 95% visual, 5% text. If text can be removed, remove it.
- The color glow IS the label in most cases -- viewers associate colors across slides
- Luminous lines, translucent surfaces, soft glows. Never flat/matte.
- Negative space is a design element, not wasted space
- Each image must work as a standalone poster AND as part of a 10-slide sequence
- The overall arc: Question (1) → Reveal (2) → Foundation (3) → Journey (4) → Platform (5) → Contrast (6) → Security (7) → Depth (8) → Pipeline (9) → Answer (10)
